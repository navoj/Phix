UnzipFindItem

Definition: include builtins\LiteZip.e

object res = UnzipFindItem(atom hzip, ZIPENTRY ze, string name, bool case_insensitive=false)
Description: Locates a file by name.

hzip: the result of a successful call to UnzipOpenFile().
ze: the result of a successful call to new_ZIPENTRY().
name: the file to find. A fatal error occurs if name exceeds MAX_PATH (=260) characters.
case_insensitive: self explanatory.

Returns ZR_OK (=0, and sets the content of ze) if found, or a string error message.
Example:
Copy 
include builtins\LiteZip.e

atom hzip = UnzipOpenFile("phix.0.7.7.1.zip")

UnzipSetBaseDir(hzip,current_dir()&"\\test")

ZIPENTRY ze = new_ZIPENTRY()
?UnzipFindItem(hzip,ze,"readme.txt") -- prints 0 (ZR_OK)
?UnzipGetFileName(ze) -- prints current_dir()&"\\test\\readme.txt"

UnzipClose(hzip)
See Also: UnzipOpenFile, ZIPENTRY, UnzipSetBaseDir, UnzipGetFileName, UnzipClose
