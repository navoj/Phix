IupSetAttributeHandle

Definition: include pGUI.e

IupSetAttributeHandle(Ihandln ih, string name, Ihandle ih_named)
Description: Instead of using IupSetHandle and IupSetAttribute with a new creative name, this function automatically creates a non conflict name and associates the name with the attribute.

It is very useful for associating images and menus.

ih: identifier of the interface element. If NULL will set in the global environment.
name: name of the attribute.
ih_named: element to associate using a name

The routine will not check for inheritance since all the attributes that associate handles are not inheritable.
See Also: IupGetAttributeHandle, IupSetAttribute, IupSetAttributes, IupSetHandle
