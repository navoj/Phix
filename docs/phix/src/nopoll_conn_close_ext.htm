nopoll_conn_close_ext

Definition: include nopoll.e

nopoll_conn_close_ext(atom conn, integer status, string reason)
Description: Close an opened noPollConn no matter its role (noPollRole).

conn: The connection to close.
status: Optional status code to send to remote side. If status is < 0, no status code is sent.
reason: The content to be sent.
See Also: nopoll_conn_close
