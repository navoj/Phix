glClearColor

Definition: include pGUI.e
include opengl.e

glClearColor(atom red, blue, green, alpha)
Description: The glClearColor function specifies the red, green, blue, and alpha values used by glClear to clear the color buffers.
Values specified by glClearColor are clamped to the range [0-1]. The default values aer all 0.
Remarks GL_INVALID_OPERATION is generated if glClearColor is executed between the execution of glBegin and the corresponding execution of glEnd.

Existing values can be retrieved via glGet with argument GL_ACCUM_CLEAR_VALUE or GL_COLOR_CLEAR_VALUE (which can likewise yield GL_INVALID_OPERATION).
See Also: glBegin, glClear, glEnd
