wdCanvasViewport

Definition: include pGUI.e

wdCanvasViewport(cdCanvas hCdCanvas, atom xmin, xmax, ymin, ymax)
Description: Configures a viewport in the canvas coordinate system to be used to convert world coordinates (with values in real numbers) into canvas coordinates (with values in integers).

The default viewport is the whole canvas (0,w-1,0,h-1). If the canvas size is changed, the viewport will not be automatically updated.
