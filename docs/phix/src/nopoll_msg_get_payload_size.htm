nopoll_msg_get_payload_size

Definition: include nopoll.e

integer len = nopoll_msg_get_payload_size(atom pMsg)
Description: Get the payload byte length stored on the provided message.

pMsg: The websocket message to get the payload from.

Returns: The payload size or -1 if it fails (only when msg is NULL).
See Also: nopoll_conn_get_msg, nopoll_conn_read
