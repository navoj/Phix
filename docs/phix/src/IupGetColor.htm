IupGetColor

Definition: include pGUI.e

integer {result, r, g, b} = IupGetColor(integer x=IUP_CENTERPARENT, y=IUP_CENTERPARENT, r=255, g=255, b=255)
Description: Shows a modal dialog which allows the user to select a color. Based on IupColorDlg.

x, y: x, y values of the IupPopup function. r, g, b: define the color selected when the dialog is shown.

Returns: a result of 1 if the OK button is pressed, or 0 otherwise. If the OK button is not pressed, the r, g and b values are as passed.
Notes: The dialog uses a global attribute called "PARENTDIALOG" as the parent dialog if it is defined. It also uses a global attribute called "ICON" as the dialog icon if it is defined.
Example: 
include pGUI.e

    IupOpen()
    ? IupGetColor(100,100,192,192,255)
    IupClose()

    {} = wait_key()
