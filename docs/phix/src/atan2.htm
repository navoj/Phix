atan2

Definition: atom res = atan2(atom y, atom x)
Description: Calculate the arctangent of a ratio.

y: the numerator of the ratio
x: the denominator of the ratio

Returns an atom, which is equal to arctan(y/x), except that it can handle zero denominator and is more accurate.
Example: 
a = atan2(10.5, 3.1)
-- a is 1.283713958
