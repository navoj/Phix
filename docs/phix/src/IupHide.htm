IupHide

Definition: include pGUI.e

IupHide(Ihandle ih)
Description: Hides an interface element. This function has the same effect as attributing value "NO" to the interface element’s VISIBLE attribute.
ih: Identifier of the interface element.
Notes: Once a dialog is hidden, either by means of IupHide or by changing the VISIBLE attribute or by means of a click in the window close button, the elements inside this dialog are not destroyed, so that you can show them again.
To destroy dialogs, the IupDestroy function must be called.
See Also: IupShowXY, IupShow, IupPopup, IupDestroy
