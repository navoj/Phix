cdText

Syntax: include pGUI.e

cdText(atom x, atom y, string text)
-- or --
cdCanvasText(cdCanvas hCdCanvas, atom x, atom y, string text)
Description: Draws a text in the position (x,y) according to the current font and text alignment. It expects an ANSI string. Can have line breaks.

cdText(x,y,t) is equivalent to cdCanvasText(cdActiveCanvas(),x,y,t).
