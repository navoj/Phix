get_file_size

Definition: atom filesize = get_file_size(string filename)
Description: Get the size of a file in bytes.

filename: the name of the file to query.

Returns an atom, the file size, or -1 if file is not found or a directory.
Comments: This function does not compute the total size for a directory, and returns -1 instead.

You will also get -1 if a wildcard is passed (see dir1 file.

Both forwardslash ('/') and backslash ('\\') are handled for all platforms.
Example: 
?get_file_size("C:/Program Files (x86)/Phix/builtins/pfile.e")
Implementation: See builtins\pfile.e (an autoinclude) for details of the actual implementation.
See Also: dir
Technicalia For compatibility with OpenEuphoria, file_length() is a simple alias of get_file_size(), and behaves identically.
Note that in OpenEuphoria file_length() returns 0 for directories, whereas in phix it returns -1.
