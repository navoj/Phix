IupUnmap

Definition: include pGUI.e

IupUnmap(Ihandle ih)
Description: Unmap the element from the native system. It will also unmap all its children.

It will NOT detach the element from its parent, and it will NOT destroy the IUP element.

ih: Identifier of an interface element.
Notes: When the element is mapped some attributes are stored only in the native system.
If the element is unmaped those attributes are lost.
Use the function IupSaveClassAttributes when you want to unmap the element and keep its attributes.
See Also: IupAppend, IupDetach, IupMap, IupCreate, IupDestroy
