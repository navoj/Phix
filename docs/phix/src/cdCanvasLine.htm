cdCanvasLine

Definition: include pGUI.e

cdCanvasLine(cdCanvas hCdCanvas, atom x1, y1, x2, y2)
Description: Draws a line from (x1,y1) to (x2,y2) using the current foreground color and line width and style. Both points are included in the line.

The procedure cdLine(x1,y1,x2,y2) is shorthand for cdCanvasLine(cdActiveCanvas(),x1,y1,x2,y2).
